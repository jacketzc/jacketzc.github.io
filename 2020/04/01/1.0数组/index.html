1.0数组

理念
1.可以存放任意类型数据的数组
2.拥有默认大小，不够时自动扩容

模块化设计理念：
制作一个最小的功能让其他的方法调用，比如说先设计一个add方法，然后再addLast方法以及addFirst等方法中调用

域：
data : E[]
size : int
1.该类的本质是一个范型数组
创建范型数组：
public Array<E>{
    private E[] data;
    public Array(int capacity)
    {
          data=(E[])new Object[capacity];
//无法直接创建范型数组！
     }
}
2.需要维护的另一个变量记录这个数组中已经存入的元素个数

构造方法：
Array(int)
Array()
//前者可以指定大小，后者使用默认大小

基本方法：
getSize
getCapacity
isEmpty

addFirst                    O(n)
add                           O(n/2)=O(n)
get

特殊方法：
void reSize(int)
//将现有的数组扩容(或者缩减)到指定的容量
private void reSize(int capacity){
E[] rs = (E[]) new Object[capacity];
/*for (int i=0;i<size;i++){
rs[i]=data[i];
}*/
System.arraycopy(data,0,rs,0,size);
data=rs;
}
//原理是复制到一个更大的(或者更小的)数组中
