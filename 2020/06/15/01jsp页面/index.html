01jsp页面

！在jsp编辑器中使用java脚本需要tomcat的运行前环境
//在maven中添加tomcat-catalina的依赖
//普通的web项目需要在构建项目时选中tomcat环境

jsp页面的本质是servlet

#在<%%>中使用java脚本
//该区域只能写java，不能写HTML
<%
Date date = new Date();
System.out.println(date);
%>
//该方法仅仅往控制台输出


#使用<%=var%>向页面输出变量
<%=date%>
//只能输出，不能写其他代码，不能在结尾写“；”
//内置对象out与之等效
out.print(student);
//out.println不输出换行
//不建议使用out

#可以配合html元素一起使用
<%
if(student.getStu_age()>20){
%>
<h1>成年</h1>
<% } else { %>
<h1>未成年</h1>
<% }
%>

#使用<%！%>声明全局变量（可以定义方法）
//在<%%>中写的代码已经卸载一个类中
//使用<%！%>表明其中的内容是全局的，在类中

#指定error页面：
1.创建一个error.jsp，在头部<%@ %>标签中添加isErrorPage="true"表明该页面是错误页，使用exception.getMessage()可以输出该错误
2.在头部标签中添加errorPage=""即可在发生java错误时将页面指向错误页面
