02.1并发模拟2（apache bench）

Apache Bench：apache附带的工具，用于测试网站的性能

ubuntu下安装：sudo apt install apache2-utils

测试方法：
ab -n 1000 -c 50 http://192.168.1.168:8080/test
//-n：一共请求的数量
//-c：同时请求的数量

结果解读：
Complete requests：总请求数量
Time taken for tests：请求总耗时
Total transferred：服务器响应数据总量（包括响应的头部数据）
HTML transferred：服务器响应的正文总量（上条减去头数据）
Requests per request：吞吐量（每秒请求数量）
//计算公式：Complete requests/Time taken for tests
Time per request（mean）：用户平均等待时间
Time per request（mean，across all concurrent requests）：服务器平均等待时间
Transfer rate：单位时间从服务器获得的请求数据量
//计算公式：Total transferred/Time taken for tests
