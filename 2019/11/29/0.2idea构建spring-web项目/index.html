0.2idea构建spring-web项目

1.创建项目时，在Spring选项中同时勾选Spring和web，同时生成web.xml文件
//这种方式创建的项目将同时引入web和spring的jar包
2.配置tomcat
    新建一个工件：
         project-structure-artifacts，点击左上角的+，新建一个“web-application：exployed”
          //工件就是最终生成的项目，通常是打包成war文件，放置到tomcat的webapps中启动
          //这一步可以省略
    点击run按钮左边的add confugurations
    在server选项卡中选择已经配置好的tomcat服务器
    在deployment中选择tomcat启动的工件


额外的操作：
1.如何构建web项目之后导入spring框架
    老方法：可以在web-WEB-INF-lib目录中放置spring的jar包
    新方法：project-structure-modules，在项目文件夹下新建一个spring模组
                     在artifacts选项卡中WEB-INF-lib目录中添加新创建的spring模组
