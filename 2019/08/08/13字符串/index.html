2019-08-08 JAVA编程思想
Word count: 400 | Reading time: 1min

13字符串

String对象不可变，任何看起来修改String值的方法实际上都是创建了一个全新的对象
//str.replaceAll()执行后str本身并没有改变，它的返回值才是改变的结果
//封装类型和String一样，在传递引用时（例如作为函数的参数），实际情况只是拷贝了一个副本，对副本的修改并不会修改原值
//空串为“”，空串也是一个字符串，而null不是空串

StringBuilder对象
在使用“+”运算符来连接字符串的时候，编译器默认创建了一个StringBuilder对象：
new StringBuilder（String）.append（String）
字符串的效率提升：如果在循环中使用“+”运算符，不如直接new一个StringBuilder对象（每次使用“+”都会创建一个StringBuilder）
StringBuilder result=new StringBuilder（）；
for（i...）{
    result.append（String）；
}
return result.toString（）；
//StringBuilder对象的reversed()方法可以反转该字符串

格式化的输出：
System.out.printf()
来源：格式化输出来自Formatter类，标准方法为.format（）
//printf（）只是约定写法

切割字符串：
.spit(String regex，int limit)
//将字符串从指定匹配位置切割，结果返回一个String[]
//可限制次数（可选）

替换字符串：
.replaceFirst（String regex,String replacement）：替换第一个
.replaceAll（String regex,String replacement）：替换所有

部分获取
.subString(int begin,int end)
//返回该字符串的子串，参数决定开始与结束的位置
//end为可选部分

开头匹配和结尾匹配
bool startsWith()
bool endsWith()

字符串比较
.equals(String)
.equalsIgnoreCase(String)
//不区分大小写
//杜绝使用“==”来比较字符串的相等性
